## Overview

*MECHCONNECT* is a digital platform built to connect Mechanical Engineering students across all academic levels (100L–500L) at FUTO. It provides a centralized space for learning resources, collaboration, project sharing, and skill development — eliminating the fragmentation that slows academic growth.

---

## Problem Statement

Mechanical Engineering students face two persistent challenges:

- *Scattered academic materials* — notes, past questions, and resources are spread across chats, drives, and individuals with no central access point.
- *Weak cross-level collaboration* — students rarely interact across year groups, leaving juniors without mentorship and seniors without fresh perspectives.

MECHCONNECT solves both.

## Features

| Feature | Description |
|---|---|
| 🖥️ Student Dashboard | Personalized interface for each student |
| 📚 Academic Resources | Centralized library of notes, materials & past questions |
| 🔧 Project Sharing | Upload, browse, and collaborate on engineering projects |
| 💬 Cross-Level Communication | Connect 100L–500L students in one space |
| ⚙️ Skill Development Hub | Guides and resources for CAD, simulation, programming & more |
| 🔮 Real-Time Collaboration (planned) | Live tools for team-based project work |
